Uwaga: aplikacja potrzebuje wszystkich plików z folderu Gotowa aplikacja.

Aby uruchomić aplikację kliknij setup.exe
Następnie wybierz opcję Zainstaluj
Aby wybrać sprawdzić pogodę wybierz przycisk start
Wpisz nazwę miasta i kliknij zapytanie.
Jeśli chcesz możesz wpisać kolejne miasto lub możesz wrócić do poprzedniego okienka klikając przycisk powrót.
Aplikacja kończy działanie w przypadku kliknięcia przycisku wyjście w menu głównym lub klikniencia X
